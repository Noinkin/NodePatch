# PatchPackages

Patch Packages

## get()

Import a package

**Parameters:**
- `name` *(string)* — Package name

## reload()

Reload a package by its name

**Parameters:**
- `name` *(string)* — Package name

## reloadAll()

Reloads all packages

## clearCache()

Clears import cache of a module

**Parameters:**
- `modulePath` *(string)* — Filepath of module
