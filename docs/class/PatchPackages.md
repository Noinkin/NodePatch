# PatchPackages

Patch Packages

## get()

Import a package

**Parameters:**

- `name` _(string)_ — Package name

## reload()

Reload a package by its name

**Parameters:**

- `name` _(string)_ — Package name

## reloadAll()

Reloads all packages

## clearCache()

Clears import cache of a module

**Parameters:**

- `modulePath` _(string)_ — Filepath of module
